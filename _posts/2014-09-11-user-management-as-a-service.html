</a>This month's meeting will take place on Monday 15th August, 7pm for 7.15pm start, at The Skiff in Brighton as usual. This month we welcome Emmanuel Ide</strong></a> back to speak, and he'll be presenting a case study on "User management as a service</strong>", illustrating the use of the Stormpath</a> PHP SDK to integrate user management into your applications. From the Stormpath website:</p>

Stormpath is the first user management and authentication service for developers. A scalable cloud-based API, Stormpath provides a common role-based access control & authentication framework that can be used across disparate applications and multiple programming languages. It also provides an on-line management console and workflows for user registration and password recovery.</p>

Stormpath virtualizes your user stores to simplify access to multiple directories from one application. Groups and roles reflect user roles and organizations, including SaaS directories, and are easy for any admin to manage. Directories, accounts, and groups are available to any connected application via our web console or REST +JSON API. Authentication occurs instantly with one simple API call.</p>
