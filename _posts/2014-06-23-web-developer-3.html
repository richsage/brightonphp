Job Description:</h2>

We are looking for an experienced developer to work in our spacious offices in central Brighton. The role will focus on backend development using primarily PHP and MySQL.</p>

The team is using lots of new and interesting technologies, such as dependency management with Composer; a bunch of Symfony 2 components; Vagrant for virtualization; one-line deployments and continuous integration.</p>

A core part of the role will be developing our powerful e-commerce platform using brand new code and exciting new technologies.</p>

The successful candidate will have:</p>

  • Experience using object-orientated PHP 5 and MySQL queries</li>
  • Solid HTML skills and an understanding of semantic mark-up</li>
    </ul>

    Our values</p>

    • We really care about quality, not only of what the client can see, but also the code we have to work with. Everything we produce is peer reviewed. We find that these code and functionality reviews are a great way of sharing knowledge and pushing up the standard. GitHub's pull requests system is integral to the way we work.</li>
    • We believe in automation; deploying our code using a one line command. Everything is handled automatically and if something goes wrong it's just one more command to roll back.</li>
    • We believe in documenting our code well, so another developers can easily understand it next year, and new developers can quickly orient themselves.</li>
    • We believe in writing tests for our code, so we can safely refactor it when we find ways to make it better; and we can keep technical debt at bay.</li>
    • We believe in modularising our code by writing small independent libraries that can be easily swapped out, and which, when combined, form a complete system. (We use Composer/Satis to manage these). No behemoth spaghetti-code here.</li>
      </ul>
